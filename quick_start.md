## 🎨 Customization Tips

### Change Colors
In `styles.css`:
```css
:root {
    --primary-color: #e74c3c;    /* Main red */
    --secondary-color: #c0392b;  /* Dark red */
    --accent-color: #f39c12;     /* Orange */
}
```

### Add Your Images
Replace image URLs in `index.html`:
```html
<img src="YOUR_IMAGE_URL_HERE" alt="Description">
```

- `+91-80808-08080` (phone)

## 🆘 Troubleshooting

### Problem: Site not loading
**Solution:** Check URL, clear cache, try different browser

### Problem: Form not working
**Solution:** Check browser console (F12), verify email config

### Problem: Images not showing
**Solution:** Verify image URLs, use HTTPS only

### Problem: Slow performance
**Solution:** Compress images, use CDN, enable caching
